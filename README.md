# Tasks

Graph-based project management app: tasks and their relationships are modeled as a directed graph.

A Python + TypeScript port of [code-red](https://github.com/floriandejonckheere/code-red) by Florian Dejonckheere (MIT), originally built with Ruby on Rails and RedisGraph.

## Stack

- **Backend**: Python, FastAPI, SQLAlchemy + Alembic (PostgreSQL), Neo4j (graph storage)
- **Frontend**: TypeScript, React, Vite, TailwindCSS, D3.js
